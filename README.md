### Starting...</h4>
#### Make a clone</h6>
```
git clone git@github.com:giovannalinda/weather-location.git
```
### install the dependencies</h6>
```
yarn
# or
npm install
```
### Usage</h3>
```
yarn dev
# or
npm run dev
```
Open http://localhost:3000 to view it in the browser.

### Technologies used</h3>
  * ReactJS
  * [OpenWeather](https://openweathermap.org/)
  * ESLint
  * EditorConfig
